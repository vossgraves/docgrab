How It Works

1

Start the Server

Run python server.py to start the backend on port 5000.

2

Paste the URL

Copy any SlideShare or Scribd document link and paste it into the input field.

3

Download Your PDF

Hit Download — the server fetches all pages, compiles them into a PDF, and gives you the file.
